eq?(*argv) {|self, result_object| ...}

Shorthand for splattr(:eq,*argv,&block)

Returns true or false based on the resulting operations for each object property comparison.

If each object property comparison is true, eq? will return true, otherwise eq? will return false.

ne?(*argv,&block)

Shorthand for splattr(:ne,*argv,&block)

Returns true or false based on the resulting operations for each object property comparison.

If each object property comparison is false, ne? will return true, otherwise ne? will return false.

splattr(operand,*argv,&block)

Returns true if the resulting operations for each object property comparison returns true, otherwise, splattr returns false.

For instance, if each object property comparison returns true and operand == :eq, splattr will return true; if each object property comparison returns true and operand == :ne, splattr will return true.
